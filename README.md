# 🌊 FinWave

> Ride the Wave of Smarter Finance

An Apple-inspired personal finance platform designed to transform financial chaos into clarity through a premium and intuitive experience.

## 🌐 Live Demo
https://finwave-delta.vercel.app/

## ✨ Features

- 🍎 Apple-inspired UI/UX
- 🖥️ macOS-style collapsible sidebar
- 💰 Income & Expense Tracking
- 📊 Spending Analytics
- 🎯 Budget Planner
- 🤖 AI Insights
- 📱 Responsive Design
- ⚡ Smooth Animations
